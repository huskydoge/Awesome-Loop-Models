# Daily Loop-Model Briefing - 2026-07-12

## Outcome

One new paper was added today: **RSF-GLLM: Bridging the Semantic Gap in Multi-Hop
Knowledge Graph QA via Recurrent Soft-Flow and Decoupled LLM Generation**
(`2607.06527`). It is LLM-related, but it is not a looped-language-model architecture:
the reused learned operator lives in the knowledge-graph reasoner that prepares
grounded evidence for a separate LLM generator.

## Added

### RSF-GLLM (`2607.06527`)

RSF-GLLM targets multi-hop knowledge-graph question answering, where useful paths can
pass through intermediate entities that share little lexical overlap with the original
question. Its Recurrent Soft-Flow module uses a GRU-guided query updater to repeatedly
propagate continuous relevance scores over the graph. Dynamic gating helps traverse
semantically dissimilar bridge nodes, while flow-sparsity regularization encourages the
soft distribution to converge toward discrete reasoning paths.

The framework then extracts and textualizes those paths to fine-tune an LLM for answer
generation. This division of labor is the important architectural detail: the recurrent
graph module performs differentiable structural reasoning, and the LLM turns its
evidence into an answer grounded in the selected graph topology.

Why it was added: the RSF module reuses a learned GRU-based query update inside one
model-side reasoning process, satisfying the collection's internal recurrence rule. It
is categorized as an application because the contribution is specialized to
knowledge-graph QA rather than a general recurrent-depth LLM design.

Main reader takeaway: the paper reports competitive results on WebQSP and Complex
WebQuestions, with better inference efficiency than more computationally expensive
LLM-based approaches. The abstract does not provide exact gains, so the result should
be read as evidence for the recurrent reasoner-plus-generator decomposition rather
than as a quantified state-of-the-art claim.

## Checked but not added

- **Hidden Decoding at Scale: Towards Untapped Latent Reasoning Capabilities in
  Language Models** (`2607.08186`) expands latent streams along sequence length while
  keeping the backbone fixed. It discusses looped and depth-recurrent Transformers but
  does not itself introduce reused internal-block recurrence.
- **Linear Attention Architectures: From Recurrent Memory to Parallel Computation**
  (`2607.07953`) studies sequence-time recurrent memory in linear attention rather than
  recurrent depth within a single reasoning process.
- **Latent Memory Palace: Test-Time Compute for Long-Horizon Control** (`2607.08724`)
  uses autoregressive latent variational inference for control, outside the strict
  reused-layer loop-model boundary.
- Agent and workflow-loop papers were excluded unless they exposed a reused learned
  internal operator; repeated model calls, planning, and verification loops do not meet
  this collection's scope rule.
