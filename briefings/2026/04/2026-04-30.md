Today’s collection update has two in-scope additions. **On Halting vs Converging in Recurrent Graph Neural Networks** is a theory-oriented paper about recurrent graph neural networks, where message passing is iterated until a convergence or halting condition is met. **Coevolutionary Continuous Discrete Diffusion** adds a language-modeling design angle: one time-conditioned model repeatedly denoises both continuous latent representations and discrete token states.

The recurrent-GNN paper fits because the computation reuses the learned graph update/message-passing operator repeatedly within one prediction process. Its main value is a clean comparison of stopping semantics: global convergence, output-level convergence, and per-vertex halting.

The CCDD paper fits because diffusion sampling is an iterative inference process: the same denoising network is reapplied across timesteps to refine an internal state. The paper is especially relevant to this collection because it explicitly proves that a continuous diffusion sampler can simulate looped-transformer rollouts, then uses joint continuous-discrete denoising to make the latent-reasoning path more trainable for language modeling.

The associative-state Universal Transformer paper remains worth watching, but it is not included as a paper entry today. It appears related to shared-depth recurrence, yet the current evidence is closer to a small pilot than a solid architecture result, so it belongs in the watchlist rather than the main collection for now.
